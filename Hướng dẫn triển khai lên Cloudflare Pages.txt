
## 🚀 Hướng dẫn triển khai lên Cloudflare Pages

### Phương án A — Upload trực tiếp (Không cần code, đơn giản nhất)

**Bước 1:** Tải file ZIP deploy:  
> File ZIP đã được tạo tại sandbox `/home/user/rung-chuong-vang-cloudflare-upload.zip` (8.7MB)  
> Hoặc dùng source backup: https://www.genspark.ai/api/files/s/yjhTTt7A → giải nén → vào thư mục `webapp/dist/` → zip lại

**Bước 2:** Đăng nhập https://dash.cloudflare.com → chọn **Pages**

**Bước 3:** Nhấn **"Create a project"** → chọn **"Direct Upload"**

**Bước 4:** Đặt tên project (ví dụ: `rung-chuong-vang`) → nhấn **"Create project"**

**Bước 5:** Upload ZIP → kéo thả hoặc chọn file `rung-chuong-vang-cloudflare-upload.zip`

**Bước 6:** Nhấn **"Deploy site"** → đợi 30-60 giây

**Bước 7:** Nhận URL dạng: `https://rung-chuong-vang.pages.dev` ✅

### Phương án B — Dùng Wrangler CLI (Cho developer)

```bash
# 1. Cài Node.js 18+ và Wrangler
npm install -g wrangler

# 2. Đăng nhập Cloudflare
wrangler login

# 3. Clone/giải nén source code
# Giải nén yjhTTt7A.tar.gz và vào thư mục webapp

# 4. Cài dependencies và build
npm install
npm run build
cp -r public/assets dist/

# 5. Tạo project và deploy
npx wrangler pages project create rung-chuong-vang --production-branch main
npx wrangler pages deploy dist --project-name rung-chuong-vang

# 6. URL: https://rung-chuong-vang.pages.dev
```

Hoàn toàn ĐƯỢC và đây là một ý tưởng RẤT XUẤT SẮC!

Việc dùng giọng thật của giáo viên để đọc câu hỏi không chỉ giải quyết triệt để lỗi TTS Tiếng Việt (do nhiều trình duyệt web không hỗ trợ giọng tiếng Việt chuẩn), mà còn mang lại sự gần gũi, thân thiện, giúp các bé mầm non hào hứng hơn rất nhiều so với giọng robot tự động.

Dưới đây là ý tưởng thiết kế và hướng dẫn chi tiết từng bước dành cho bạn:

1. Ý tưởng thiết kế vào Game
Thay vì phụ thuộc vào hàm window.speechSynthesis (đọc text tự động) dễ sinh lỗi, chúng ta sẽ viết thêm chức năng đọc file âm thanh thủ công.

Dữ liệu câu hỏi: Mỗi câu hỏi trong bộ 30 câu Tiếng Việt sẽ được bổ sung thêm 2 trường dữ liệu mới:
audio_question: File thu âm đọc câu hỏi và 3 lựa chọn (phát ngay khi hiện câu hỏi).
audio_answer: File thu âm đọc đáp án đúng (phát khi bảng đáp án hiện ra cuối giờ).
Logic game: Khi game load đến câu hỏi, code sẽ tự động ưu tiên tìm file .mp3 này để phát. Nếu không có file .mp3, thì nó mới dùng TTS (như một phương án dự phòng).
2. Hướng dẫn chi tiết cách thu âm (Dành cho Giáo viên)
Để game có chất lượng tốt nhất, bạn chuẩn bị theo các bước sau nhé:

Bước 1: Cách đọc để thu âm Bạn cần thu âm ở nơi yên tĩnh. Với mỗi câu hỏi (tổng cộng 30 câu), bạn thu thành 2 file riêng biệt:

File số 1 (Đọc Đề bài):
"Ngôi nhà cần vật liệu gì để xây?" (Nghỉ 1 giây) "Một: Gỗ và gạch." (Nghỉ 0.5s) "Hai: Tuyết và băng." (Nghỉ 0.5s) "Ba: Lá cây."

File số 2 (Đọc Đáp án - Khi có kết quả):
Trọng giọng hào hứng: "Tuyệt vời! Đáp án đúng là số một: Gỗ và gạch!"

Bước 2: Đặt tên file thật chuẩn Nên lưu file ở định dạng .mp3 (hoặc .m4a cũng được). Khi lưu, hãy đặt tên theo quy tắc sau để code dễ dàng nhận diện tự động mà không cần phải gõ lại code từng file:

File đề bài: cau1.mp3, cau2.mp3, cau3.mp3 ... đến cau30.mp3.
File đáp án: cau1_ans.mp3, cau2_ans.mp3, cau3_ans.mp3 ... đến cau30_ans.mp3.
Bước 3: Đưa file vào dự án Sau khi thu âm xong toàn bộ (tối đa 60 file), bạn hãy tạo một thư mục trống trong dự án theo đường dẫn: public/assets/audio/vi/ Và copy toàn bộ các file mp3 đã thu âm thả vào trong đó.

3. Cách Tích hợp vào Code như thế nào?
Bạn không cần phải lo về việc viết chữ nào vào mã nguồn cả. Khi nào bạn thu âm và copy bỏ các file mp3 vào thư mục xong, bạn chỉ cần nhắn cho tôi một câu: 👉 "Tôi đã thu âm xong, hãy cập nhật code để đọc audio nhé"

Lúc đó, tôi sẽ sửa lại file src/index.tsx để:

Thêm máy phát nhạc (Audio object) ẩn đằng sau giao diện.
Sửa lại hàm speakQuestion() để nó fetch file cau[id].mp3.
Sửa lại hàm revealAnswer() để nó fetch file cau[id]_ans.mp3.
Vẫn giữ nguyên mọi hiệu ứng hình ảnh xịn xò của project.
Bạn thấy ý tưởng và hướng dẫn này đã đủ cho bạn bắt đầu khâu ghi âm chưa? Nếu cần, tôi có thể điền trước khung logic vào code (chưa có mp3) để lúc bạn bỏ mp3 vào là nó tự chạy luôn từ bây giờ!
